Internet

The Internet is a worldwide network of connected computers. It allows devices to share information with each other.

how are the internet and web different?

The Internet is the network, and the Web is something that uses that network. The Web cannot work without the Internet.

ASCII and unicode

ASCII is a basic system for English letters and symbols. Unicode supports many languages and special characters.

what is UTF-8

UTF-8 is a way to store Unicode characters on computers. It is widely used on the web.

URL

A URL is the address of a website or page on the Internet. It tells the browser where to go.

Domain name

A domain name is the easy-to-read name of a website, like google.com. It represents an IP address.

HTTP

HTTP is a rule used to send web pages to your browser.It is not secure

HTTPS

HTTPS is the secure version of HTTP. It protects data by encrypting it.

IP address

An IP address is a number that identifies a device on a network,Computers use it to communicate.

DNS

DNS turns domain into IP addresses and it helps browsers find websites

W3C Recommendations

W3C Recommendations are rules for building websites correctly. They help pages work the same across browsers.

XML, XHTML, HTML, HTML5

XML stores data, and HTML creates web pages. XHTML is stricter than HTML. HTML5 is the newest version with more features.
